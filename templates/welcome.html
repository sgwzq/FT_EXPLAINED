Welcome to this web application.
It was designed in addition to the script given for the publication: Basics of Fourier Transform Applied to NMR Spectroscopy: An Interactive Open-source Web Application.

This app aims at explaining how the NMR signal is processed by the Fourier Transform to generate a spectrum, but NMR spectrum interpretation (chemical shift and multiplicity) is not addressed here.

If you are running the app for the first time, we advise you to read the explanation messages. Afterward, you can turn them off using the corresponding button in the main screen.

The app needs your browser to accept cookies and to allow javascript execution.
